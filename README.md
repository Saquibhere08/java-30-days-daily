# ☕ 30-Day Java Beginner Practice Challenges

A simple **30-day challenge roadmap** to help beginners practice Java
every day and build strong programming fundamentals.

------------------------------------------------------------------------

## 🚀 How to Use

-   Solve **one challenge per day**
-   Try solving it **before looking for solutions**
-   Practice writing the code yourself
-   Push your solutions to this repository

------------------------------------------------------------------------

# 📅 Challenge Plan

## 🟢 Week 1 --- Basics (Variables, Input, Math)

**Day 1:** Print `Hello, Java World!`

**Day 2:** Create variables for **name, age, and city** and print them
in a sentence.

**Day 3:** Take **two numbers from user input** and print their sum.

**Day 4:** Calculate the **area of a rectangle**\
`area = length × width`

**Day 5:** Convert **Celsius to Fahrenheit**\
`F = (C × 9/5) + 32`

**Day 6:** Find the **largest of two numbers**.

**Day 7:** Create a **simple calculator** (+ − × ÷).

------------------------------------------------------------------------

## 🟡 Week 2 --- Conditions (if, else, switch)

**Day 8:** Check if a number is **even or odd**.

**Day 9:** Check if a number is **positive, negative, or zero**.

**Day 10:** Find the **largest of three numbers**.

**Day 11:** Check if a year is a **leap year**.

**Day 12:** Convert **marks to grade**.

    90–100 = A
    80–89 = B
    70–79 = C

**Day 13:** Use **switch** to print the **day of week**.

    1 = Monday

**Day 14:** Create a **menu program**:

    1. Pizza
    2. Burger
    3. Coffee

------------------------------------------------------------------------

## 🔵 Week 3 --- Loops

**Day 15:** Print numbers **1 to 100**.

**Day 16:** Print a **multiplication table**.

**Day 17:** Find the **sum of numbers from 1 to N**.

**Day 18:** Calculate **factorial of a number**.

**Day 19:** Reverse a number.

    1234 → 4321

**Day 20:** Check if a number is a **palindrome**.

    121 → palindrome

**Day 21:** Print star pattern

    *
    **
    ***
    ****

------------------------------------------------------------------------

## 🟣 Week 4 --- Arrays & Methods

**Day 22:** Store **5 numbers in an array** and print them.

**Day 23:** Find the **largest number in an array**.

**Day 24:** Find the **average of array numbers**.

**Day 25:** Count **even and odd numbers** in an array.

**Day 26:** Write a **method to calculate square**.

**Day 27:** Write a method to **check prime number**.

**Day 28:** **Sort an array** (bubble sort).

**Day 29:** **Search a number** (linear search).

**Day 30:** 🎓 **Mini Project -- Student Grade Calculator** - Input
marks - Calculate average - Display grade
